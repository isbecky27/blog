波蘭巧克力棉花糖

「你可以不吃，但是一定要請你朋友吃」系列！（？） 據說波蘭的巧克力棉花糖很有名，這篇就來介紹 Wawel 品牌的 Królewskie Mleczko - Śmietankowe (奶油口味)。 不負責任翻譯：Royal Creamy Marshmallows 皇家奶油棉花糖 ( 因為有人覺得很難吃而開始四處發放，所以開始認真記錄大家吃完的心得 ) 個人心得 不確定是巧克力還是棉花糖的味道，總之有股我不喜歡的神秘味道，有點熟悉但又講不出是甚麼的味道。裡面的棉花糖吃起來不像棉花糖，是一咬就斷的口感，很酷值得一試，但真的就是"一試"。 要不是因為發這篇文查了翻譯，真的不知道原來它是奶油口味 !? 為了查證，又打開了一片吃了幾口，對不起.. 我真的找不到奶油味在哪裡 QAQ 其他試吃員的評價 像太妃糖的味道 還可以，沒有到難吃 可以試一次 結果統計 好吃 普通 不好吃 0 2 3
